Cortex Council

A multi-agent AI orchestration system where specialized LLM agents collaborate to solve complex tasks. Built with LangGraph for stateful agent flows, each agent owns a domain — research, reasoning, coding, critique — and they debate toward a final answer. Runs fully local via Ollama.
